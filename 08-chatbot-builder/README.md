# Chatbot Builder

Generates a working Python chatbot using the Claude API. Clean, minimal code with a conversation loop — ready to run and hand off to a client.

## Install

Copy `chatbot-builder.md` to your `.claude/commands/` folder.

## Requirements

Client needs Python and an Anthropic API key.

## Usage

```
/chatbot-builder [describe what the chatbot is for]
```

## Example

```
/chatbot-builder A FAQ chatbot for a dentist clinic that answers questions about appointment booking, pricing and procedures
```

Claude will generate the full Python script with comments.
