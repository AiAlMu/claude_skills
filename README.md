# Claude Skills for Job Escape

22 ready-to-use Claude Code skills for AI freelancers. Clone, copy the skill you need to your `.claude/commands/` folder, and use it as a slash command.

## How to install a skill

1. Copy the `.md` file from any skill folder
2. Paste it into your project's `.claude/commands/` folder (create it if it doesn't exist)
3. Open Claude Code and type `/skill-name your input here`
