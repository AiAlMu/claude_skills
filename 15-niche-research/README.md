# Niche Research

Researches any industry to find where AI and automation create the most value. Gives you pain points, specific workflow ideas, tools, positioning, and pricing ranges.

## Install

Copy `niche-research.md` to your `.claude/commands/` folder.

## Usage

```
/niche-research [industry or niche you want to target]
```

## Example

```
/niche-research Real estate agents
```

Use this before pitching a new industry so you walk in knowing their problems better than they do.
