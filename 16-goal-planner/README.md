# Goal Planner

Breaks down a big life goal into a 90-day plan with monthly targets and this week's specific actions. Realistic, not motivational.

## Install

Copy `goal-planner.md` to your `.claude/commands/` folder.

## Usage

```
/goal-planner [describe your goal]
```

## Example

```
/goal-planner I want to earn $3,000/month from freelancing within 3 months. I currently have no clients and I'm just finishing the Job Escape course.
```

Claude will give you a 90-day roadmap with weekly actions and the obstacles you'll actually face.
