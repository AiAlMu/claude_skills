# Decision Helper

Helps you think through a difficult decision. Breaks down both options with reasons and risks, asks the one key question that cuts through the noise, and gives a direct recommendation.

## Install

Copy `decision-helper.md` to your `.claude/commands/` folder.

## Usage

```
/decision-helper [describe the decision you're facing]
```

## Example

```
/decision-helper Should I quit my job now to freelance full-time or wait until I have 3 clients? I have $4,000 in savings, one steady client paying $800/month, and a job I hate.
```

Claude will give you a structured analysis and a clear opinion — not a wishy-washy "it depends."
