# Portfolio Case Study

Turns a project description into a professional portfolio case study. Structured as Problem → Solution → Results. Client-facing and results-first.

## Install

Copy `portfolio-case-study.md` to your `.claude/commands/` folder.

## Usage

```
/portfolio-case-study [describe the project you completed]
```

## Example

```
/portfolio-case-study I built an n8n workflow for a real estate agency that automatically sends personalized follow-up emails to leads within 5 minutes of them filling a contact form. Before this they were responding manually, 1-2 days later.
```

Claude will write a 250-word case study ready for your portfolio.
