# Fiverr Gig

Writes a complete Fiverr gig — title, description and 3 packages (Basic / Standard / Premium) — for any AI or automation service.

## Install

Copy `fiverr-gig.md` to your `.claude/commands/` folder.

## Usage

```
/fiverr-gig [describe the service you want to offer]
```

## Example

```
/fiverr-gig I want to sell n8n automations that connect CRMs to email tools for small businesses
```

Claude will output a full gig ready to paste into Fiverr.
