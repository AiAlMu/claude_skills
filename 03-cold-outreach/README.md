# Cold Outreach

Writes a short, personalized cold email or DM to a potential client. No generic openers. Leads with their pain, ends with one clear CTA.

## Install

Copy `cold-outreach.md` to your `.claude/commands/` folder.

## Usage

```
/cold-outreach [describe who you're messaging and why]
```

## Example

```
/cold-outreach Small e-commerce store owner, they post a lot on Instagram manually every day, I want to offer them an AI content automation system
```

Claude will write a message under 120 words, ready to send.
