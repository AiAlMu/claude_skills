# API Integration

Writes a clean Python script to connect two APIs. Handles auth, the core data flow, and basic error handling. Beginner-readable code.

## Install

Copy `api-integration.md` to your `.claude/commands/` folder.

## Usage

```
/api-integration [describe what you want to connect and how]
```

## Example

```
/api-integration Get new leads from Typeform → create a contact in HubSpot CRM with their name, email and form answers
```

Claude will write the full Python script with comments.
