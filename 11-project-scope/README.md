# Project Scope

Creates a Scope of Work document with deliverables, timeline, what's NOT included, and a price. Protects you from scope creep and looks professional.

## Install

Copy `project-scope.md` to your `.claude/commands/` folder.

## Usage

```
/project-scope [describe the project a client wants]
```

## Example

```
/project-scope Client wants me to build an automation that takes job applications from their website, scores them with AI, and sends the top ones to their Slack channel
```

Send the output to the client before starting any work.
