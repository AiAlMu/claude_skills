# Budget Plan

Creates a personal monthly budget with income, fixed expenses, variable costs, a savings target, and where to cut if needed.

## Install

Copy `budget-plan.md` to your `.claude/commands/` folder.

## Usage

```
/budget-plan [describe your income and main expenses]
```

## Example

```
/budget-plan I earn $2,200/month. Rent $800, phone $40, gym $30, groceries roughly $300, transport $150, eating out way too much probably $400, subscriptions ~$60.
```

Claude will output a clean budget table and tell you where the money is going.
