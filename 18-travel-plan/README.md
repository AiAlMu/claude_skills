# Travel Plan

Plans a trip with a day-by-day itinerary, accommodation recommendations, budget breakdown, and money-saving tips for the destination.

## Install

Copy `travel-plan.md` to your `.claude/commands/` folder.

## Usage

```
/travel-plan [destination, dates, budget, travel style]
```

## Example

```
/travel-plan Bali, 10 days in July, budget $1,500 total, first time there, mix of relaxing and exploring, not a party person
```

Claude will plan your full trip including where to stay, what to do each day, and how much to expect to spend.
