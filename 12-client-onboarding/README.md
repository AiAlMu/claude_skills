# Client Onboarding

Generates a welcome message and onboarding checklist for a new client. Makes you look organized from day one and collects everything you need to start.

## Install

Copy `client-onboarding.md` to your `.claude/commands/` folder.

## Usage

```
/client-onboarding [describe the project and client]
```

## Example

```
/client-onboarding New client, small marketing agency, I'm building them an n8n workflow that auto-posts their blog content to LinkedIn and Twitter
```

Send the output as your first message after the client confirms the project.
