# Automation Audit

Analyzes any business and finds the top 5 automation opportunities — with tool recommendations, time saved estimates, difficulty ratings, and pricing guidance.

## Install

Copy `automation-audit.md` to your `.claude/commands/` folder.

## Usage

```
/automation-audit [describe the business]
```

## Example

```
/automation-audit A freelance accountant who handles 30 clients. They manually send invoices, follow up on payments, collect receipts via email, and update a spreadsheet with all transactions.
```

Great to use before a client proposal — shows them the value before they hire you.
