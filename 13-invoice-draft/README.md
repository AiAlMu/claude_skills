# Invoice Draft

Creates a professional invoice with line items, totals, payment terms, and a thank-you note. Ready to send or paste into your invoicing tool.

## Install

Copy `invoice-draft.md` to your `.claude/commands/` folder.

## Usage

```
/invoice-draft [describe the work done and payment details]
```

## Example

```
/invoice-draft Client: TechStart Ltd. Work: n8n automation setup for lead routing - 8 hours at $75/hr. Plus $50 for n8n cloud setup. Payment due in 7 days via bank transfer.
```

Claude will output a formatted invoice ready to send.
