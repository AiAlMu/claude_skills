# System Prompt Builder

Creates a well-structured system prompt for any AI assistant or chatbot. Defines the role, tone, rules, and boundaries.

## Install

Copy `system-prompt.md` to your `.claude/commands/` folder.

## Usage

```
/system-prompt [describe what the AI assistant should do]
```

## Example

```
/system-prompt A customer support bot for an online clothing store. It should answer questions about orders, returns and sizes. It should never discuss competitors or give refunds directly.
```

Claude will write a production-ready system prompt you can paste into any AI tool.
