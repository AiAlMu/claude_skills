# Discovery Call

Prepares a script and question list for a first call with a potential client. Helps you sound confident, ask the right questions, and close toward a next step.

## Install

Copy `discovery-call.md` to your `.claude/commands/` folder.

## Usage

```
/discovery-call [describe the client and type of project]
```

## Example

```
/discovery-call Restaurant owner who wants to automate customer review responses and reservation confirmations
```

Claude will give you a call script with opening, key questions, and a closing move.
