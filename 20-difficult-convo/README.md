# Difficult Conversation

Helps you prepare for any hard conversation — with a client, boss, partner, or family member. Gives you an opening, key points, how to handle pushback, and what to avoid.

## Install

Copy `difficult-convo.md` to your `.claude/commands/` folder.

## Usage

```
/difficult-convo [describe the situation]
```

## Example

```
/difficult-convo I need to tell a client that the project will take 2 extra weeks because the requirements changed halfway through. They're already impatient and I'm worried they'll be angry.
```

Claude will prepare you so the conversation goes as well as possible.
