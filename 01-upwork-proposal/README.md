# Upwork Proposal

Generates a tailored Upwork proposal for any job posting. Personal, concise, and focused on the client's result — not your CV.

## Install

Copy `upwork-proposal.md` to your `.claude/commands/` folder.

## Usage

```
/upwork-proposal [paste the job posting or describe it]
```

## Example

```
/upwork-proposal Looking for someone to automate our lead follow-up emails using AI. We get 50+ leads a day and manually write each email. Budget $300-500.
```

Claude will write a ready-to-send proposal under 200 words.
