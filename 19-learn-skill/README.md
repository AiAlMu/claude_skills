# Learn Skill

Builds a step-by-step learning roadmap for any skill — with phases, best resources, and one concrete action to take today.

## Install

Copy `learn-skill.md` to your `.claude/commands/` folder.

## Usage

```
/learn-skill [skill you want to learn]
```

## Example

```
/learn-skill Spanish from zero, I have 30 minutes per day, want to be conversational in 6 months
```

Works for any skill — language, coding, design, music, copywriting, anything.
